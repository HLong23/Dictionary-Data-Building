# Dictionary Data Building

Ứng dụng từ điển dòng lệnh hỗ trợ quản lý và tra cứu từ vựng.

## Tính năng

- `lookup <word>` - Tra cứu từ
- `lookupshow <word>` - Tra cứu và hiển thị chi tiết
- `define <word>` - Định nghĩa từ
- `drop <word>` - Xóa từ
- `export` - Xuất dữ liệu từ điển
- `upload pronounce <word> <file-path>` - Upload file phát âm
- `help` - Xem hướng dẫn
- `exit` - Thoát

## Cách sử dụng

```bash
javac src/*.java
java src/Main
```
